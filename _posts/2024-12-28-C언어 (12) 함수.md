# **C언어 (12) 함수**
> 함수<br>

### 포인터 변수를 통한 1차원 배열 접근
배열의 이름은 배열의 시작 주소와 같다고 했다.<br>
그리고 배열의 주소에 +1, +2를 하면 주소는 배열의 요소 크기만큼 증가한다.<br>
```c
#include <stdio.h>
int main() {
    int arr[5] = { 1, 2, 3, 4, 5 };
    double ar[5];
    printf("%p %p %p\n", arr, arr+1, arr+2);
    printf("%p %p %p\n", ar, ar+2, ar+4);
    return 0;
}
```
